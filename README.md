# Reeyc
环岛高铁WiFi综合管理平台

## Server setup (in /src/server)
```
npm install --安装服务端依赖
```

## Server setup (in /src/server)
```
npm run api-serve --启动Mock服务器
```

## Project setup
```
npm install --安装项目依赖
```

### Compiles and hot-reloads for development
```
npm run serve --启动项目
```

### Compiles and minifies for production
```
npm run build --构建项目
```
